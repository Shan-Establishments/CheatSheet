HTML Cheatsheet
Property Name Description
height Height Property Changes the height of an HTML element using values such as px, em, % or auto
width Width Property Changes the width of an HTML element using values such as px, em, % or auto
font-family Font-Family Property Changes the style of the webpages font
font-weight Font-Weight Property Changes the boldness of the webpages font
font-size Font-Size Property Changes the size of the webpages font
line-height Line-Height Property Determines the amount of space from the top and bottom the the text
background-color Background Colour Property Determines the background colour of the HTML element
color Colour Property Determines the colour of the HTML element
background-image Background Image Property Adds a background image behind HTML elements
padding Padding Property Adds space between HTML elements and the its borders
margin Margin Property Adds space outside of the elements borders
