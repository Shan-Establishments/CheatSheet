CSS Cheatsheet
Tag Name Description
<!DOCTYPE> Document Type Tag Notifies the browser on the version of HTML the document is using e.g <!DOCTYPE html>
<html> </html> HTML Tag Contains all the html elements apart from <!DOCTYPE>
<head> </head> Head Tag The head tag contains metadata (data about data): the title <title>, the links <link>, the script <script>, the styles <style> and the meta character set <meta charset="UTF-8">
<body> </body> Body Tag Contains the content of the document including images, videos, information etc.
<main> </main> Main Tag Contains the main content of the document
<nav> </nav> Navigation Tag Contains navigation links used to direct users to a different sections of the webpage or a different webpage entirely.
<article> </article> The Article Tag Specifies a specific content within the document
<section> </section> Section Tag Represents a section of the content
<aside> </aside> Aside Tag Provides additional information about the main content.
<footer> </footer> Footer Tag Contains copyright, navigation and/or authorised information.
